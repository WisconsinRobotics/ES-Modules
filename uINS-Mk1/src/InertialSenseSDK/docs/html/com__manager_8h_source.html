93 /*
94 Packet Overview
95 
96 Byte
97 0 Packet start byte
98 1 Packet indo: ID (mask 0x1F) | reserved bits (mask 0xE)
99 2 Packet counter (for ACK and retry)
100 3 Packet flags
101 
102 // packet body, may or may not exist depending on packet id - packet body is made up of 4 byte or 8 byte values.
103 4-7 Data identifier
104 8-11 Data length
105 12-15 Data offset
106 16-19 Data start
107 (n-8)-(n-5) Last piece of data
108 // end data
109 
110 n-4 Reserved
111 n-3 Checksum high byte
112 n-2 Checksum low byte
113 n-1 Packet end byte
114 */
