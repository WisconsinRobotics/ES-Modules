Packet flags Bit 0 : unset means big endian, set means little endian format Bit 1 : unset means no valid communication received yet, set means valid communication received Bit 2 : unset means no more related packets available, set means additional packet(s) available related to this packet Bit 3 : unset means perform swap, set means do not swap
